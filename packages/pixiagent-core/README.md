# @pixiagent/core

The Pixi Agent Core package provides the base agent framework for the monorepo.

## Guides

- [Agent interrupt and timeout guide](./docs/agent-interrupt-timeout.zh-CN.md)

## What is included

- `Agent` lifecycle and history management
- plugin registration and lifecycle hooks
- `AgentMessage` and `AgentState` types

## Development

Run from the repository root:

```bash
pnpm install
pnpm test
pnpm build
```
